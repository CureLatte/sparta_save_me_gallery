# 🎨 유화제작 서비스(9해줘갤러리)

<br>

## 소개
<p style="font-size: 1.2rem;">
9해줘갤러리는 미술관을 컨셉으로 하여,체험(유화제작서비스:AI)을 하고 
체험한 결과물을 전시(혹은 저장:CRUD) 하고, 후기(comment:CRUD) 를 남길 수 있는 웹 서비스 이며, 이번 프로젝트에서는, 학습 내용의 복습 및 적용을 주 목적으로 정하였습니다.
</p>
